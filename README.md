# Rustlings — My Rust Learning Journey

A personal walkthrough of the [rustlings](https://github.com/rust-lang/rustlings) exercises to sharpen my Rust programming skills and deepen my understanding of the language.

## About This Repo

I'm using this repo to brush up on Rust and build stronger knowledge of core concepts — ownership, borrowing, lifetimes, error handling, traits, iterators, and more. Each lesson folder contains my completed solutions as I work through the exercises.
